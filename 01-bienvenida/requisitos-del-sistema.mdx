# Requisitos del Sistema

## Para Usarlo

-   **Navegador web actualizado**: Chrome, Firefox, Edge o Safari
-   **Conexión a internet estable**: Recomendado mínimo 5 Mbps
-   **Pantalla**: Mínimo 1280x720 píxeles (recomendado 1920x1080)

## Dispositivos Compatibles

-   ✅ Computadoras de escritorio (Windows, macOS, Linux)
-   ✅ Laptops
-   ✅ Tablets (funcionalidad completa)
-   ⚠️ Celulares (funcionalidad limitada, mejor usar la app)
